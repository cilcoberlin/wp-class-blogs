== Description ==

A theme designed to support the Oberlin CILC's class blogs suite.  Its main
function is to display a list of student posts from all blogs on the site on
the front page in columns sorted by the user who has most recently posted.

== Dependencies ==

For the Bentham theme to function optimally, you need to have the Oberlin CILC's
class blogs suite enabled.  Information and the source for this suite can be
found at http://languages.oberlin.edu/cilc/projects/class-blogs/.
