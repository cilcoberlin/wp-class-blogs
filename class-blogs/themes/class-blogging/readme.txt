== Description ==

A theme that takes advantage of the functions provide by the Class Blogs plugin
to display a list of student posts from all blogs on the site on the front page
in columns sorted by the user who has most recently posted.

== Dependencies ==

For the Class Blogging theme to function properly, you must have the Class Blogs
plugin installed and activated.
